Laboratory facilities

The department has well furnished labs equipped with latest configuration systems to satisfy the computing requirements for various activities in the different academic disciplines. The systems are connected to high end servers with 8 Mbps internet connection toprovide internet fesility for students and staff. The laboratories are equiped with latest high end systmes and new software.

The important laboratories include:

  • Data Structures and Algorithms Laboratory

  • File Structures Laboratory

  • OOP Laboratory with C++ and JAVA

  • Database Applications Lab

  • System Programming Lab.

  • Microprocessor Laboratory

  • Networks Laboratory

  • Project Work/ Research Lab

  • Electronics and Logic design Laboratory

  • Software Testing Lab

Major Equipments:

    The department has 129 systems and 3 servers with latest configurations with UPS facility for all the systems, power Supplies of different voltage ranges, Function Generator, Digital IC trainer kit, Linear IC kit, Signal generator, Interfaces, Analog and Digital IC Tester, Decade Resistance Box, Decade Inductance Box, Decade Capacitance Box, Dual Trace Oscilloscope, Digital Multi-meters & Digital CROs in EC-LD Lab. HP LaserJet Printers, HP inkjet Printers Scanners, 8086 software package, LCD Projector, Turbo C/C++ compiler, Symantec antivirus, Windows XP, Windows 7, and Fedora 15 etc.
